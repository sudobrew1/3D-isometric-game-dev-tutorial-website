# Isometric Game Tutorials

Tutorials for building pre-rendered isometric game art — modeling and baking
tiles in Blender, then getting them into Godot and Unity.

**Live site:** Coming soon!

## What's here

- `src/` — site source: markdown content, layout, CSS, JS
- `src/_data/nav.js` — sidebar structure
- `src/_includes/base.njk` — the page layout
- `eleventy.config.js` — build config and plugins

## Running locally

    npm install
    npx @11ty/eleventy --serve

Then open http://localhost:8080.

## Built with

Eleventy 3.x, Nunjucks templates, CSS (no framework),
Prism for syntax highlighting.
